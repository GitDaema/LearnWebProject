# 원광대학교 AI 웹정보서비스 (Wonkwang University AI Web Information Service)

원광대학교 재학생들을 위한 AI 기반 종합 웹정보서비스(포털) 시스템입니다. 자연어 질문을 통해 학사 정보를 손쉽게 조회할 수 있는 **지능형 AI 챗봇 인터페이스**와, 교내 주요 서비스로 빠르게 이동할 수 있는 **일반 포털 네비게이션**이 결합된 하이브리드 포털 데모 프로젝트입니다.

## 🚀 주요 기능

### 1. AI 학사 지원 챗봇 (홈 화면)
* **자연어 질의 분석:** "이번 학기 성적 알려줘", "인공지능개론 강의계획서 확인해줘", "휴학 신청은 어디서 해?" 같은 자연어 질문의 의도를 분석합니다.
* **Function Calling 도구 연동:** Gemini API의 도구 정의를 이용해 질문의 의도(Intent)와 매개변수를 분류한 뒤 알맞은 카드 UI를 출력합니다.
* **팩트 기반 가이드 문장 생성:** 시스템 프롬프트에 모의 학사 데이터베이스를 포함하여, AI가 실제 데이터에 기반한 정확한 사실만을 답변으로 작성하도록 통제합니다.
* **로컬 데모 모드 (Fallback):** API 키가 없거나 할당량이 초과된 경우에도 정규식과 키워드 매칭을 이용한 자체 로컬 규칙 분석기를 구동하여 중단 없는 데모 시연을 보장합니다.

### 2. 일반 포털 메뉴 (사이드바 및 개별 페이지)
* **개인정보관리:** 인적사항 정보 조회 및 종합정보시스템 비밀번호 변경 외부 링크 연동
* **학적관리:** 휴·복학 신청(파라미터 `?type=leave` 대응), 자퇴 신청(`?type=dropout` 대응) 페이지 및 신청 상태 이력 조회
* **시간표관리:** 컴퓨터공학과에 개설된 모든 과목의 상세 리스트를 조회하고 학년별, 이수구분(전필/전선/기전/교필/교선)별 필터링 및 검색이 가능한 시간표 대시보드
* **수업관리:** 교과목별 상세 강의계획서(주차별 일정 포함) 조회 및 수강신청관리(외부 링크), 수강신청조회, 수업평가 기능 제공
* **전자출결관리:** 교과목별 출결 현황 조회(출석/지각/결석/공결 시간 및 출석률) 및 공결 신청 내역 현황판 제공
* **성적관리:** 학기별 취득학점, 평점평균(GPA), 상세 이수과목 성적 리스트 조회
* **등록관리:** 등록금 납부 고지서 조회(실 납부액 계산 및 가상 계좌 확인) 및 등록금 환불 신청 상세 내역(`?tab=refund`) 제공

## 🛠 기술 스택

* **Frontend Framework:** Next.js 16 (App Router / Turbopack)
* **Language:** TypeScript
* **Styling:** Vanilla CSS & Tailwind CSS (어두운 배경의 글래스모피즘 테마 디자인 반영)
* **Icons:** Lucide React
* **AI Model:** Google Generative AI (Gemini 2.5-flash / 3.1-flash-lite / 3.5-flash 로테이션 폴백)

## 💻 시작하기

### 1. 의존성 패키지 설치
```bash
npm install
```

### 2. 환경 변수 설정
프로젝트 루트 폴더에 `.env.local` 파일을 생성하고 발급받은 Gemini API 키를 입력합니다.
```env
GEMINI_API_KEY=your_gemini_api_key_here
```
> **Note:** API 키가 없는 경우, 로그인된 헤더 화면 및 채팅 창에 "로컬 데모 모드 (Fallback)" 경고와 배지가 나타나며, 내장된 로컬 분석기를 통해 주요 기능이 데모 동작합니다.

### 3. 개발 서버 실행
```bash
npm run dev
```
개발 서버가 켜지면 브라우저를 열고 [http://localhost:3000](http://localhost:3000)으로 접속하여 시연을 진행합니다.
